# Project Drumstick

A simple web application that plays drum sounds when you press corresponding keys on the keyboard or click the on-screen buttons.

## Setup

1.  Clone this repository to your local machine.
2.  Open the `index.html` file in your web browser.

That's it! No special installation is required.

## Usage

You can play the drum kit in two ways:

*   **Clicking the buttons:** Use your mouse to click on the drum buttons on the page.
*   **Pressing keys:** Press the keys on your keyboard to trigger the corresponding drum sounds.

Here is a list of the available drum sounds and their corresponding keys:

| Key | Sound       |
|-----|-------------|
| s   | Tom 1       |
| a   | Tom 2       |
| n   | Tom 3       |
| t   | Tom 4       |
| k   | Snare       |
| u   | Crash       |
| m   | Kick Bass   |

## Project Structure

*   `index.html`: The main HTML file that provides the structure of the web page.
*   `styles.css`: Contains the styles for the drum kit interface.
*   `index.js`: Contains the JavaScript code that handles user interactions (clicks and key presses) and plays the drum sounds.
*   `sounds/`: This directory contains all the drum sound audio files in `.mp3` format.
*   `images/`: This directory contains any images used in the project.
*   `README.md`: This file, providing information about the project.
